Adds a grid of products with a click and expand on the Storefront Homepage template.

== Description ==


This plugin is built to work only for the [Storefront theme](https://wordpress.org/themes/storefront).

**How to use:**

After installation go to Appearance => Customize => Gridder Products Gallery and change the values to fit your needs.

**Notes:**


== Installation ==

**Install via the WordPress Dashboard:**

1. Login to your WordPress Dashboard.
2. Navigate to Plugins, and select add new to go to the "Add Plugins" page.
3. In the right side, enter "Storefront Gridder" in the search input bar, and hit your enter key.
4. Click install, and wait for the plugin to download. Once done, activate the plugin.


**Install via FTP:**

1. Extract the zip file, login using your ftp client, and upload the storefront-gridder folder to your /wp-content/plugins/ directory
2. Login to your WordPress Dashboard.
3. go to plugins and activate "Storefront Gridder" plugin.



== Frequently Asked Questions ==

**Will this plugin work for themes other than Storefront?**
Unfortunately, No.

**I've activated the plugin, where can I access the settings?**
The settings for this plugin can be found in the Customize page under Appearance. In that page, find the section named "Gridder Products Gallery".

**Will this work on mobile or is it responsive?**
Is designed to be responsive. We've also added an option in the customizer to hide the Gridder Products Gallery entirely when viewing it on mobile.
